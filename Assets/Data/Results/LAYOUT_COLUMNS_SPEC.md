# 배치(layout) 요약 컬럼 스펙 — `combined_timeseries_*_layout.csv`

기존 시계열 주행 CSV(`combined_timeseries_XXX-YYY.csv`)의 각 행 **맨 끝에 7개 컬럼**을 덧붙인 파일이
`*_layout.csv` 이다. 이 7개 값은 **정적 배치로 결정되는 상수** → 한 주행(케이스) 안의 모든 행에서 동일하게 반복된다.

- 조인 키: CSV의 **`LayoutID`**(= `caseNNN`) → `Assets/Data/Cases/caseNNN.json`
- 생성 스크립트: `augment_layout.py`
- ⚠️ 파일명 라벨과 내부 케이스는 어긋나 있음(예: `001-050.csv` 안에는 **case080~129**). **반드시 `LayoutID`로 조인**.

## 좌표계 · 단위 (공통)

- **적재함 로컬(bed-local) 좌표**: `x=폭(좌−/우+)`, `y=위`, `z=길이(후−/전+)`. 원점 = 적재함 바닥 중심.
- **단위: 실축 SI** — 길이 `m`, 질량 `kg`, 관성모멘트 `kg·m²`.
  - ⚠️ 주의: 기존 `TotalMassKg` 컬럼은 Unity 시뮬 배율(×100)이 걸린 값이다.
    아래 배치 컬럼은 **실축**(JSON `totalMass` 스케일, 예: 0.86kg)이라 서로 스케일이 다르다.

## 추가된 7개 컬럼

| 컬럼 | 내용 | 출처 / 식 | 기준 |
|---|---|---|---|
| `CogX` | 적재물 무게중심 x (폭) | JSON `cog.x` (사전 계산됨) | bed 원점 |
| `CogY` | 적재물 무게중심 y (높이) | JSON `cog.y` | bed 바닥 |
| `CogZ` | 적재물 무게중심 z (길이) | JSON `cog.z` | bed 원점 |
| `MaxHeightM` | 적재 최고 높이 | JSON `maxHeight` | bed 바닥면=0 |
| `InertiaXX` | 롤축(x, 주행 좌우 기울기축) 관성모멘트 | 아래 식 | **적재물 CoG** 기준, bed축 정렬 |
| `InertiaYY` | 요축(y, 수직축) 관성모멘트 | 아래 식 | 〃 |
| `InertiaZZ` | 피치축(z, 길이축) 관성모멘트 | 아래 식 | 〃 |

`CogX/Y/Z`, `MaxHeightM`는 JSON에 이미 계산돼 있던 값을 그대로 옮긴 것이다.

> 참고: `적재물 개수`, `총질량`은 이미 기존 컬럼 `CargoCount`, `TotalMassKg`에 있어 중복 추가하지 않았다.
> (실축 총질량이 필요하면 JSON `totalMass` = `TotalMassKg` ÷ 100.)

## 관성모멘트(InertiaXX/YY/ZZ) 계산 기준

**정의:** 적재물 전체의 관성텐서를 **적재물 무게중심(CoG) 기준**, **적재함 축(x,y,z) 정렬**로 구한 뒤,
대각성분 3개를 보고한다. 단위 `kg·m²`. (요청대로 3축 모두, 기준은 화물 CoG.)

**모델링 가정:**
- 각 화물 = **균일밀도 직육면체**로 근사. 치수는 카탈로그 `sizeX/Y/Z_cm`의 경계상자(bounding box).
  파이프·코일·포대도 동일하게 경계상자 직육면체로 근사(간소화).
- 질량 = 카탈로그 `massKg`(실축).
- 화물 회전(`localEuler`)은 Unity `Quaternion.Euler` 규약(적용순서 Z→X→Y)으로 반영.

**식 (화물 i에 대해):**

1. 자체 관성(자기 중심, body축) — 균일 직육면체:
   ```
   Ixx = m/12 · (dy² + dz²)
   Iyy = m/12 · (dx² + dz²)
   Izz = m/12 · (dx² + dy²)         (dx,dy,dz = 화물 치수[m])
   ```
2. 회전 정렬: `I_world = R · diag(Ixx,Iyy,Izz) · Rᵀ`   (R = localEuler 회전행렬)
3. 평행축 정리로 적재물 CoG 기준 이동:
   ```
   d = (화물 위치 − 적재물 CoG)
   I_world += m · ( |d|²·E₃ − d⊗d )
   ```
4. 전 화물 합산 → 총 관성텐서. 대각성분이 `InertiaXX/YY/ZZ`.

**해석 힌트:** `InertiaXX`(롤축)가 클수록 좌우 기울임(전복 방향)에 대한 회전관성이 커
같은 토크에서 롤 각가속이 작아진다. 전복 예측의 주요 후보 피처.
